# MCGA - Make Coffee Great Again ☕

一款为咖啡爱好者打造的微信小程序，记录冲煮参数、评价咖啡品质。

## 功能特色

- **微信账号登录** — 首次使用需登录，登录状态缓存 30 天，过期自动重新登录
- **多用户数据隔离** — 每位用户只能访问自己的历史数据
- **咖啡日志列表** — 首页显示最近 5 条记录，点击「显示更多」加载更多（每次 10 条）
- **添加/编辑记录** — 记录豆子信息、冲煮参数、环境参数及品质评分
- **详情页** — 参数网格展示 + 品质条形图，一目了然
- **统计分析** — 冲煮方式分布、平均评分、趋势汇总
- **云端同步** — 数据存储在微信云数据库，跨设备同步

## 记录内容

### 豆子信息
- 咖啡豆名称、产地、烘焙度

### 冲煮参数
- 冲煮方式（手冲、意式、冷萃）
- 磨豆机型号、研磨度、粉量、水量、水温、萃取时间
- 自动计算粉水比
- 滚轮选择器调参，精确到 0.5，默认显示上次输入值

### 环境参数
- 当日气温、湿度、备注

### 品质评分（1-5 分）
- 香气、酸质、甜度、醇厚度、余韵、综合

## 技术栈

- **前端：** 微信小程序（WXML / WXSS / JS）
- **后端：** 微信云开发（Serverless）
- **数据库：** 微信云数据库（文档型）
- **云函数：** `coffeeLogFunctions` — 登录 + 增删改查 + 统计

## 用户登录

- 首次打开小程序进入登录页，点击「使用微信账号登录」
- 登录成功后状态缓存至本地，30 天内免登录
- 超过 30 天未使用则自动清除登录状态，需重新登录
- 所有数据通过 OpenID 关联到用户，每人只能访问自己的记录

## 快速开始

1. 使用微信开发者工具打开项目
2. 在云开发控制台配置云环境
3. 上传 `coffeeLogFunctions` 云函数
4. 开始记录你的咖啡 ☕
